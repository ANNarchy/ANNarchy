****************************************
Class Dendrite
****************************************

A dendrite is an ensemble of connections (or synapses) between a subset of a population (called the presynaptic population) and a postsynaptic neuron. These ensamble of connections can be modified through adding or suppressing synapses to allow structural plasticity.

The Python Dendrite class allows to access or modify connectivity information via the ``get_variable`` and ``get_parameter`` functions.
