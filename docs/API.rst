***********************************
Library reference guide
***********************************

This chapter provides documentation for the different methods and classes defined in the main Python module ANNarchy. For an easier usage, ANNarchy uses an object oriented approach to represent the neural network. After defining `neurons <API/Neuron.html>`_ and `Synapses <API/Synapse.html>`_, the user build up his network consisting of `populations <API/Population.html>`_ interconnected by `projections <API/Projection.html>`_.
